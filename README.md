# WordPress Envioment to made Themes

## Install Docker:
> See the link:
[dl.docker.com](https://docs.docker.com/get-docker/)
## Run localy:

> Run commands:

```
cd wordpress-docker-envioment
```

```
sudo docker-compose -f "docker-compose.yml" up -d --build
```

> Open the link:
[localhost:8080](http://localhost:8080)
